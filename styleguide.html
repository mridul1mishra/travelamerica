Travels Americas — Color Style Guide

Reference for all color decisions across travelsamericas.com  ·  Last updated June 2026

Brand palette

Brand Red
#B22234
Logo, scrolled header text, CTA buttons, accent borders. Primary brand identifier.
Brand Navy
#0d1b2a
Header background (default, unscrolled), footer, hero overlay. Anchors the American travel aesthetic.
Navy Soft
#1b263b
Section dark backgrounds, card overlays, dark text on light surfaces.
Gold Accent
#facc15
Nav link hover on dark backgrounds, star ratings, highlight badges. Never use as a primary button color.

Action & interactive

Action Blue
#2563eb
Primary buttons, links, focus rings, icon fills. The single canonical interactive color.
Action Blue Dark
#1d4ed8
Button hover, active states, pressed indicators. Always paired with #2563eb.
Action Blue Light
#3b82f6
Inline link color on white backgrounds. Text links only — not buttons.
Blue Tint
#dbeafe
Info callout backgrounds, tag chips, selected state backgrounds.

Text

Text Primary
#111827

Text Muted
#64748b

Text Disabled
#94a3b8

Usage rules

Do

  • Use #B22234 for the scrolled header text and the single "Start planning" CTA per page
  • Use #2563eb for all interactive elements — links, buttons, focus rings
  • Use #facc15 only on dark backgrounds (#0d1b2a, #1b263b)
  • Keep navy (#0d1b2a) for full-bleed dark sections — header, footer, hero overlays
  • Pair status colors with a border-left accent, not full background fills

Don't

  • Don't use #7e2b7c purple anywhere — it has no brand meaning
  • Don't mix #1a73e8, #0070f3, #0364a0 with #2563eb on the same page
  • Don't use #facc15 gold as a button background — fails contrast on white
  • Don't use #B22234 for error states — reserve it for brand use only
  • Don't place #374151 text on anything darker than #f7f9fc
